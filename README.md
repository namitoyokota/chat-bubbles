# chat-bubbles.namitoyokota.com

-   **Framework**: [Angular](https://angular.io/)
-   **Deployment**: [Vercel](https://vercel.com)

## Codebase

-   `app/*` - Page content
-   `assets/*` - Static assets used on the site

## Running Locally

```shell
$ git clone https://github.com/namitoyokota/chat-bubbles.git
$ cd chat-bubbles
$ yarn install
$ yarn start
```
